# neomediavalist.github.io/libvalues/

The LibertarianValues Quiz measures and analyzes various aspects of libertarian viewpoints over the course of 60 questions, with five axes of values. LibertarianValues is based upon [LibertarianValues](https://polittest.github.io/libertarianvalues/auth/).
